Hexo 诞生之初

Hexo是台湾的大牛tommy351在大学时期开始写的作品，缘由是觉得 Octopress 生成文档的速度太慢！然后就基于 Node.js 自己写了一个 Hexo，真会玩！

请看故事 >> Hexo 颯爽登場！

Node.js & npm

Hexo 是基于 Node.js 的，所以在开始使用 Hexo 之前，肯定要安装好 Node.js 的环境才可以使用 Hexo 的，而 npm 是包含在 Node.js 的安装包中的。

请自行下载 >> Node.js

Git

Git 不是必须的，如果有 VPS 或者云服务器或一切可以存放网站的服务的话。Git 主要是用于将网站上传到 Github 或 Gitcafe 等源代码托管网站的。

也请自行下载 >> Git

Github（Page）配置

如果需要的话，请先注册 Github 账号，然后进行一下操作，创建一个仓库：

  1. 新建一个名称为 <用户名>.github.io 的 Repository
  2. 在新建的 Repository 的主界面，点击右侧的 Settings，在设置界面下方，点击即可开启 Github Page 功能
  3. 过一会，就可以通过 <用户名>.github.io 访问了

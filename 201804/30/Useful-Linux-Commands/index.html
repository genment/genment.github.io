软链接

某一运行在 VPS 上的项目，需要使用 forever 来后台运行，但是使用 npm install forever -g 安装完后，无论在哪输入 forever ，都提示 -bash: forever command not found。

后来，发现原来是被安装在 nodejs 的目录（上面👆解压出来的目录）里面了，路径大概是 ~/node-v8.9.1-linux-x86/lib/node_modules/forever/bin/forever。

由于懒得添加 $PATH，不知道哪里来的灵感，突然想到了使用软链接来创建全局可用的 forever 命令。遂 whereis node 看了一下路径，也把 forever 放到同一个位置里了。
